# Task Completion Checklist

When completing a code task, ensure:

1. **Run Tests**: `cargo test` - all tests must pass
2. **Format Code**: `cargo fmt` - ensure consistent formatting
3. **Lint Code**: `cargo clippy` - fix any warnings
4. **Build Check**: `cargo build` - ensure it compiles
5. **Update Documentation**: Update README.md if adding features
6. **Test Coverage**: Run `cargo tarpaulin --out Html` if measuring coverage
